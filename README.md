# INFO698-Fall2025-Capstone
“Predictive Modeling of University Student GPA Using Machine Learning Techniques”
(Research-Based Project)

Repository for INFO Capstone Project This repository contains the code and notebooks for the INFO 698 Capstone Project: an analysis of cummulative Term GPA data. This repository and its contents were created by Vinoth Kumar and Pradeep Jondhale.

Our project repository comprised of-1.Python implementation code for Model-1-ML Linear regression model , Model-2-Ridge regression ML Model ,and Model-3-Deep Learning ANN. 2.Model1 and Model2 Output files in Excel Format .3. Powerpoint presentation giving implementation of Models and theory behind it.
